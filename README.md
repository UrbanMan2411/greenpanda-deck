# GREEN PANDA — презентация для партнёров (deck)

Полноэкранная интерактивная презентация-сайт для B2B-партнёров бренда **GREEN PANDA** (ТМ ООО «КубаньБытХим»). 12 слайдов 16:9 с плавными переходами, навигацией стрелками/колесом/тапом и экспортом в PNG/PDF.

🟢 **Прод:** https://deck-rho-seven.vercel.app/

## Стек
React 18 · Vite · Framer Motion (анимации) · Tailwind CSS · lucide-react (иконки) · html2canvas + jsPDF (экспорт). Деплой на Vercel.

## Запуск
```bash
npm install
npm run dev        # http://localhost:5173
npm run build
npm run preview
```

## Деплой
Подключён к Vercel (auto-deploy при `git push` в `main`). Ручной: `npx vercel --prod`.

## Управление
- **Стрелки ← →**, колесо мыши, свайп (планшеты) — листать слайды.
- Клик по точкам прогресс-бара внизу — переход на слайд.
- Кнопки сверху по центру: **PNG** (текущий слайд) и **PDF** (вся презентация одним файлом — html2canvas → jsPDF, по слайду на страницу).

## Структура
```
public/                 panda.png, bg.png, sku/*.jpg (фото SKU для слайда 8)
src/
  App.jsx               навигация, экспорт PNG/PDF, монтаж слайдов
  index.css             базовые стили + .slide-frame/.slide-canvas (16:9 letterbox)
  components/
    SlideShell.jsx      общий «каркас» слайда (шапка с лого, фон, футер)
    Brand.jsx           логотип-вордмарк в углу
    Nav.jsx             стрелки, прогресс-бар, кнопки экспорта
  slides/
    index.js            РЕЕСТР слайдов (порядок, id, заголовки)
    S01_Title.jsx … S13_Contacts.jsx
  data.js               ВСЕ тексты/цифры/контент слайдов
```

## Как редактировать
- **Тексты и цифры** — почти всё вынесено в `src/data.js` (рынок, аудитория, ассортимент, роадмап, контакты и т.д.). Меняй там — слайды подхватят.
- **Добавить / убрать / переставить слайд** — `src/slides/index.js` (массив `SLIDES`). Нумерация (`01/12…`) считается автоматически через пропсы `num/total`.
- **Стиль слайда** — соответствующий `SXX_*.jsx`. Общая шапка/футер — `components/SlideShell.jsx`.
- **Контакты** — объект `CONTACTS` в `data.js`.
